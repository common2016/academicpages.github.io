fixest包简介

陈普

2022-10-20

1.基本的用法

使用fixest包进行固定效应。为说明用法，先看一个包含了多个固定效应的数据集。

library(fixest)
data("trade")
head(trade)
#>   Destination Origin Product Year  dist_km    Euros
#> 1          LU     BE       1 2007 139.5719  2966697
#> 2          BE     LU       1 2007 139.5719  6755030
#> 3          LU     BE       2 2007 139.5719 57078782
#> 4          BE     LU       2 2007 139.5719  7117406
#> 5          LU     BE       3 2007 139.5719 17379821
#> 6          BE     LU       3 2007 139.5719  2622254

现在考虑如下回归，

\[ \ln Euros_{i,j,p,t} = \beta\cdot \ln dist_{ij} + \gamma_i^{des}+\gamma_j^{ori}+\gamma_p^{pro}+\gamma_t^{year}+\varepsilon_{i,j,p,t} \]

包含了四个固定效应，每个固定效应下面都包含有多个“水平”。该模型估计语法如下，

graols <- feols(log(Euros)~log(dist_km)|Origin+Destination+Product+Year, trade)
summary(graols)
#> OLS estimation, Dep. Var.: log(Euros)
#> Observations: 38,325 
#> Fixed-effects: Origin: 15,  Destination: 15,  Product: 20,  Year: 10
#> Standard-errors: Clustered (Origin) 
#>              Estimate Std. Error  t value   Pr(>|t|)    
#> log(dist_km) -2.16988   0.154283 -14.0642 1.1898e-09 ***
#> ---
#> Signif. codes:  0 '***' 0.001 '**' 0.01 '*' 0.05 '.' 0.1 ' ' 1
#> RMSE: 1.74337     Adj. R2: 0.705139
#>                 Within R2: 0.219322

从中可以看到，

  • 每个固定效应包含的“水平”，比如Origin有15个。
  • 模型默认在第一个固定效应上进行聚类标准准误估计。

聚类标准误可以更加多样的形式进行控制，比如利用summary函数的参数vcov，

summary(graols, vcov = 'twoway')
#> OLS estimation, Dep. Var.: log(Euros)
#> Observations: 38,325 
#> Fixed-effects: Origin: 15,  Destination: 15,  Product: 20,  Year: 10
#> Standard-errors: Clustered (Origin & Destination) 
#>              Estimate Std. Error  t value   Pr(>|t|)    
#> log(dist_km) -2.16988   0.171367 -12.6621 4.6802e-09 ***
#> ---
#> Signif. codes:  0 '***' 0.001 '**' 0.01 '*' 0.05 '.' 0.1 ' ' 1
#> RMSE: 1.74337     Adj. R2: 0.705139
#>                 Within R2: 0.219322

这就是对前面两个固定效应进行聚类标准误估计。你也可以在第三个固定效应上进行聚类，如

summary(graols, vcov = ~ Product)
#> OLS estimation, Dep. Var.: log(Euros)
#> Observations: 38,325 
#> Fixed-effects: Origin: 15,  Destination: 15,  Product: 20,  Year: 10
#> Standard-errors: Clustered (Product) 
#>              Estimate Std. Error t value  Pr(>|t|)    
#> log(dist_km) -2.16988   0.077077 -28.152 < 2.2e-16 ***
#> ---
#> Signif. codes:  0 '***' 0.001 '**' 0.01 '*' 0.05 '.' 0.1 ' ' 1
#> RMSE: 1.74337     Adj. R2: 0.705139
#>                 Within R2: 0.219322
# summary(graols, cluster = "Product") # 效果一致
# summary(graols, cluster = ~Product)

或者在三个维度上聚类，

summary(graols, vcov = ~ Origin + Destination + Product)

3.交互固定效应

两个原因可能想做交互固定效应。一是简单看看两个固定效应的联合,比如\(firm\times country\)。二是考虑可变斜率，比如在DID中经常提到的个体的时间趋势。下面的代码分别演示两种目的。

先构造一个数据集，

base = iris
names(base) = c("y", paste0("x", 1:3), "fe1")
# Create another "fixed-effect"
base$fe2 = rep(letters[1:5], 30)
head(base)
#>     y  x1  x2  x3    fe1 fe2
#> 1 5.1 3.5 1.4 0.2 setosa   a
#> 2 4.9 3.0 1.4 0.2 setosa   b
#> 3 4.7 3.2 1.3 0.2 setosa   c
#> 4 4.6 3.1 1.5 0.2 setosa   d
#> 5 5.0 3.6 1.4 0.2 setosa   e
#> 6 5.4 3.9 1.7 0.4 setosa   a

交互固定效应，

est_comb = feols(y ~ x1 | fe1^fe2, base)

4.因子变量的用法

因子变量可在公式中以i()形式引入，该函数包含两个参数ref和keep，分别用来指定基准组和要保留的组。

data(airquality)
res_i1 = feols(Ozone ~ Solar.R + i(Month), airquality)
res_i2 = feols(Ozone ~ Solar.R + i(Month, ref = 8), airquality)
res_i3 = feols(Ozone ~ Solar.R + i(Month, keep = 5:6), airquality)

etable(res_i1, res_i2, res_i3, dict = c("6" = "June", "Month::5" = "May"), 
       order = c("Int|May", "Mon"))
